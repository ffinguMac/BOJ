# 🚀 Algorithm Study Repository

백준 온라인 저지와 프로그래머스에서 풀이한 알고리즘 문제들을 정리한 레포지토리입니다.

## 📌 소개

이 레포지토리는 [백준 허브(BaekjoonHub)](https://github.com/BaekjoonHub/BaekjoonHub) 확장 프로그램을 통해 자동으로 업로드되는 알고리즘 문제 풀이 모음입니다.

## 📁 레포지토리 구조

```
BOJ-1/
├── 백준/
│   ├── Bronze/      # 브론즈 티어 문제
│   ├── Silver/      # 실버 티어 문제
│   ├── Gold/        # 골드 티어 문제
│   └── Platinum/    # 플래티넘 티어 문제
└── 프로그래머스/
    ├── 0/           # 레벨 0 문제
    ├── 1/           # 레벨 1 문제
    └── 2/           # 레벨 2 문제
```

각 문제 폴더에는 다음이 포함됩니다:
- 문제 풀이 코드 파일 (`.cc`, `.cpp`, `.c`, `.py` 등)
- 문제 설명 README 파일

## 💻 사용 언어

- C++
- C
- Python

## 🛠️ 백준 허브 사용법

1. [백준 허브 확장 프로그램](https://github.com/BaekjoonHub/BaekjoonHub) 설치
2. GitHub 토큰 설정
3. 백준에서 문제를 풀고 제출하면 자동으로 커밋 및 푸시됩니다!

## 📊 통계

- **백준**: Bronze, Silver, Gold, Platinum 티어 문제 풀이
- **프로그래머스**: 레벨 0, 1, 2 문제 풀이

## 🎯 목표

- 꾸준한 알고리즘 문제 풀이를 통한 실력 향상
- 다양한 알고리즘 패턴 학습
- 문제 해결 능력 개선

## 📝 참고

이 레포지토리는 백준 허브 확장 프로그램을 통해 자동으로 관리되며, 문제를 풀 때마다 자동으로 커밋됩니다.
